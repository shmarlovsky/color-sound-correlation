1. Извлекаем кадры из видео
2. Кадры переводим из RGB в HSV
3. Строим 2D диаграмму для Hue и Saturation

4. Извлекаем аудио
5. Делим на фреймы
6. Накладываем окна на фреймы
7. Для каждого фрейма считаем спектр
8. Для каждого спектра считаем BarkBands
9. BarkBands делим на 3 части (для Red, Green, Blue значений цвета)
10. Полученный цвет переводим в HSV и строим 2D диаграмму
